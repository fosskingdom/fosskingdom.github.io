FOSS Projects:

Predictor:

Predictor is a free and open source desktop application for predicting USD/INR exchange rate and 24K e-Gold price in India. Predictor is a machine learning algorithms that builds a model based on sample given data, known as training data, in order to make gold price predictions or decisions without being explicitly programmed to do so.

  • It is a supervised machine learning algorithm to predict the future data.
  • It uses linear regression for predicting the future trends.
  • Datasets are provided by various vendors like Google, MMTCPAMP, etc.
  • It can predict values for the next 365 days.

Installation for Linux:

  • sudo apt update;
  • sudo apt install git python3-tk python3-venv xtitle -y;
  • git clone https://github.com/fosskingdom/Predictor.git;
  • cd ./Predictor/;
  • python3 -m venv ./venv/;
  • source ./venv/bin/activate;
  • chmod +x ./install.sh;
  • bash ./install.sh

Source: https://github.com/fosskingdom/Predictor.git
